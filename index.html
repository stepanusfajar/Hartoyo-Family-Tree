Getting Started

This app needs to be connected to Firebase before it can be used.

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Google Sign-In and create a Cloud Firestore database
  3. Open js/app.js and replace the Firebase config with your project's details
  4. Save and refresh this page

Need help? Ask the person who set this up, or follow the README.
